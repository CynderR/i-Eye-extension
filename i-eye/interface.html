i eyeSaving two eyes at a time!

  • How to use:

    Simply navigate to a bright page and press ctrl+q, or click on the eye-icon and then press Invert page.

    The exclude button:

    Can be used to prevent the script from running on a certain page, or to only run the script on a certain page.

    Useful if a site is for the most part white, but they have a page that is a darker color scheme. If the url has post data, you can type the path to the page you want excluded.

    You can also use the exclude button on sites that are not inverted. This will only invert the selected page.
